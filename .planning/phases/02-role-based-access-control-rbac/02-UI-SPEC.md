## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `oklch(100% 0 0)` / `oklch(15% 0 0)` | Background, surfaces (white/black) |
| Secondary (30%) | `oklch(95% 0 0)` / `oklch(25% 0 0)` | Cards, sidebar, nav |
| Accent (10%) | `oklch(45% 0.25 260)` | Blurple accents |
| Destructive | `oklch(60% 0.2 20)` | Destructive actions only |

Accent reserved for: Primary CTAs, active active selection indicators.
