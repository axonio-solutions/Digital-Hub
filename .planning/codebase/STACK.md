# Tech Stack

## Core Technologies
- **Language:** TypeScript 5.9.3
- **Runtime Environment:** Node.js (Vite environment)
- **Framework:** React 19.2.4
- **Meta-Framework:** TanStack Start (`@tanstack/react-start`) with Vinxi
- **Routing:** TanStack Router (`@tanstack/react-router`)
- **Data Fetching:** TanStack Query (`@tanstack/react-query`)
- **Build Tool:** Vite 7.3.1
- **Database ORM:** Drizzle ORM (`drizzle-orm` and `drizzle-kit`)

## Styling & UI
- **CSS Framework:** Tailwind CSS v4 (`@tailwindcss/vite`)
- **Component Primitives:** Radix UI (`@radix-ui/react-*`), vaul, sonner
- **Animation:** Framer Motion, `@visx/*` (Charting/Visualization), `tw-animate-css`
- **Form Handling:** TanStack Form (`@tanstack/react-form`), React Hook Form (`react-hook-form`), Zod for validation

## Development & Tooling
- **Package Manager:** pnpm
- **Linter/Formatter:** ESLint, Prettier
- **Testing:** Vitest, React Testing Library
- **Icons:** `lucide-react`, `@tabler/icons-react`
- **Internationalization:** i18next, react-i18next
